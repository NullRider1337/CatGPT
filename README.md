# CatGPT

Ein experimentelles neuronales Netzwerk, das versucht, Katzenantworten zu generieren.  
Inspirationsquelle: meine eigene Katze, die konstant auf meiner Tastatur sitzt.

CatGPT ist **nicht intelligent**, **nicht nützlich**, aber **sehr flauschig**.

## Features

- Generiert „katzenlogische“ Antworten
- Kennt 17 vordefinierte Miaus (Science!)
- Hat ein trainiertes JSON-„Modell“
- CLI mit Stimmungsoptionen
- 100% wissenschaftlich fragwürdig
